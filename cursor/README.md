### Extensions
* Add the `.vscode/extensions.json` file to `.vscode/` folder in the project root.
* Press `Ctrl+Shift+X` and search for `@recommended` extensions.
* Install the extensions you're interested in from the workspace recommendations.

### Workspace Settings
* Add the `.vscode/settings.json` file to `.vscode/` folder in the project root.

### User Settings
* Press `Ctrl+,` and switch to the JSON format.
* Use the `settings.json` file.

### Shortcuts
* Press `Ctrl+M Ctrl+S` and switch to the JSON format.
* Use the `keybindings.json` file.

### Projects
* Press `F1` and write `Project Manager: Edit Projects`.
* Use the `projects.json` file.
